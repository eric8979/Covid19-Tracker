Thanks for coming by!! 🐳
This is my last project without a framework (used semantic-ui.com but only used for side menubar).
The purpose is to get Covid-19 related information on my taste and practice my JS. I designed this website dynamic and somewhat usable. Check out what I used in this project and description of each menu down below. Thanks again! 😄😄


    Tools & API & functions

  • Designed menubar with CSS framework semantic-ui.com
  • Fetched API from rapidapi.com - COVID-19 API
  • Used iframe to fetch map from https://www.trackcorona.live/
  • Used import/export - for JS file arrange
  • Used async/await - for fetching API

    Menu Description

  • STATISTICS(STATS)
    • Watch increase of number of infection by country live. You can also get list of countries.
  • HISTORY
    • Get a history of covid-19 by country. You can get daily number increase, total number and more of infected and deaths.
  • GLOBE
    • Watch how wide spread the covid-19 is. The map is from trackcorona.live website. For original website, click the link!
